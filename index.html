Apps

Web Apps I've built using frameworks like Laravel and Angular.


  • Camp Fluid
    Developed

    Camp Fluid is an internal application that Fluid uses to manage clients and projects. I've integrated the Harvest, Trello and Slack API's to send notifications and create resources in each of those services. Built using the Laravel PHP framework and Foundation Front End framework.

  • InsureSpan
    Developed Designed

    InsureSpan is a customer portal for users to login and manage their insurance policies. Built using Angular and Foundation, this app also consumes an API delivered from a .NET backend.

  • TheMathCo (AcerPlacer)
    Developed

    A hybrid WordPress/Laravel site that required a small scale API for a front end questionnaire that would post answers back to the database immediately, so as not to lose any information obtained by a user.

  • Allmoxy
    Developed Designed

    Allmoxy needed a customer support/ticketing system. This application uses Laravel to manage user accounts, tickets, and FAQ's while also leveraging Ajax to return tickets that match a user's search query.
